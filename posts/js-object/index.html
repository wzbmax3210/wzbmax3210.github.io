JS 对象基本用法

November 14, 2019

声明对象的两种用法

1.简写写法

let obj = {a: 1}

2.正式写法

let obj = new Object({a: 1})

如何删除对象的属性

delete obj.xxx
delete obj['xxx']

将属性值设为undefined并不等同于删除属性

用xxx in obj来判断obj是否含xxx属性，包括原形属性 也可以用obj.hasOwnProperty来判断，只判断非原型属性

如何查看对象的属性

  1. 查看自身所有的属性用Object.keys(obj)，它将返回对象自身属性名的数组

  2. 查看自身+公有属性（prototype）用console.dir(obj)

如何修改或增加对象的属性

  1. 直接赋值

    let obj = {name: 'abc'}
obj.name = 'xxx' // name是字符串
obj['name'] = 'xxx' //name是字符串
obj['na' + 'me'] = 'xxx'

let key = 'name'
obj[key] = 'xxx' //使用变量属性时要用中括号的写法

  2. 批量赋值 使用Object.assign(obj, {a: 1, b: 2})
